# Tables

Tables are tabular representations of data with optional sorting, filtering, and pagination.

## Scope

A table definition may describe:

- table identity,
- columns,
- rows,
- sorting behavior,
- filtering behavior,
- pagination behavior,
- row selection,
- empty state, and
- loading or error state.

The specification does not require a specific table component, data-source protocol, virtual-scroll implementation, or framework component.
