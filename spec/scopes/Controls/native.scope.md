# Native

Native controls are standard browser, framework, or runtime capabilities that support the user interface without requiring a custom OpenUI control definition.

## Scope

Native controls may include:

- scroll bars,
- font size, type, and family behavior,
- color schemes,
- focus handling,
- pointer and keyboard input behavior, and
- other platform-provided presentation or interaction features.

Native behavior is usually provided through HTML, CSS, JavaScript, the browser, or the selected framework. The specification identifies the concept without requiring a specific implementation mechanism.
