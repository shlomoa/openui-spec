# Table

The Table family covers the HTML5 standard tags that present tabular data: `table`, `tr`, `th`, and `td`. Each tag is a leaf following the [leaf scope template](../../template.scope.md).

## Objects

- [table](table.scope.md): the tabular data container.
- [tr](tr.scope.md): a table row.
- [th](th.scope.md): a header cell.
- [td](td.scope.md): a data cell.

## Boundaries

These are native HTML5 tags; their contracts are drawn from the HTML standard and recorded technology-independently. Table objects follow the shared [scope folder and attribute category rules](../../scope.md).
