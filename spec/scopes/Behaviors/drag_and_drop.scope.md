# Drag and drop

Drag and drop is a behavior that allows users to move or associate elements by dragging them and dropping them on valid targets.

## Scope

A drag-and-drop behavior may describe:

- draggable item identity,
- valid drop targets,
- drag start, drag over, drop, and cancel behavior,
- accepted item types,
- ordering or reordering rules,
- disabled state, and
- accessible alternatives for pointer-based dragging.

The specification does not require a specific drag-and-drop library, pointer event implementation, keyboard model, or framework directive.
