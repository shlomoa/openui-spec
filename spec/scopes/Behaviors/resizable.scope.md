# Resizable

Resizable is a behavior that allows users to change the size of an element within a page or view.

## Scope

A resizable behavior may describe:

- resizable element identity,
- resize direction or handles,
- minimum and maximum sizes,
- default size,
- resize start, update, and end behavior,
- constraints imposed by parent layout, and
- accessible alternatives for pointer-based resizing.

The specification does not require a specific resize observer, pointer-event handler, layout engine, or framework directive.
