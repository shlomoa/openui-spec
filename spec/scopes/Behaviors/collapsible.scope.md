# Collapsible

Collapsible is a behavior that allows users to collapse and expand elements within a page or view.

## Scope

A collapsible behavior may describe:

- collapsible element identity,
- expanded or collapsed state,
- default state,
- trigger control,
- content region,
- transition expectations, and
- accessible state exposure.

The specification does not require a specific animation, disclosure widget, accordion implementation, or framework component.
