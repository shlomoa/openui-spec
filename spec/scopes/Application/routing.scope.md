# Routing

Routing defines how an application resolves navigation intents or locations to
application content.

## Scope

A routing definition may describe:

- route identity,
- route path or matching pattern,
- target page or view,
- route parameters,
- default or fallback route behavior, and
- active route state.

Routing is an application-level concept. The specification does not require a
particular router implementation, URL strategy, file-system convention, or
framework-specific route configuration format.
