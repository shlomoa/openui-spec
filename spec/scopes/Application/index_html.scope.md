# index.html

`index.html` defines the application host document and static bootstrap
metadata.

## Scope

An `index.html` definition may describe:

- document title,
- host or mount element,
- language and direction metadata,
- viewport metadata,
- icon links,
- stylesheet links,
- script entry points, and
- static application metadata.

The specification describes the host document contract without requiring a
specific framework bootstrap API, bundler, development server, or file emission
strategy.
