# favicon.ico

`favicon.ico` defines the application icon asset used for browser and shell
identity.

## Scope

A favicon definition may describe:

- the canonical icon asset path,
- icon purpose,
- supported sizes or variants,
- media or color-scheme variants, and
- fallback behavior when a preferred icon is unavailable.

The specification identifies the asset contract. It does not require a specific
build-tool asset pipeline, generated output location, or browser integration
mechanism.
