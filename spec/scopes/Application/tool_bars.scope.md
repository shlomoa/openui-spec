# Tool bars

Tool bars define application-level command surfaces for frequently used actions.

## Scope

A tool bar definition may describe:

- action identity,
- user-visible labels,
- optional icons,
- action grouping,
- placement or ordering,
- enabled, disabled, or hidden state, and
- overflow behavior for constrained layouts.

Tool bars expose application commands without prescribing a specific rendering
component, responsive breakpoint model, or command handler implementation.
