# OpenUI Specification

This folder documents the generated OpenUI framework specification. The master JSON specification is `/openui.json`, and the markdown files below document the same hierarchy in prose.

## Roots

- **Library Catalog Root** — discovered from `docs/controllibraries.md` and `library.js`; used to enumerate public libraries, dependencies, types, interfaces, controls, and elements.
- **Metadata Grammar Root** — discovered from `ManagedObjectMetadata.js` and `ElementMetadata.js`; used to normalize properties, aggregations, associations, events, defaults, renderer linkage, and drag-and-drop semantics.
